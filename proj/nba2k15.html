About the Game

NBA 2K15 features approximately 5,000 new animations, new defensive AI, shooting systems, new team-specific play sets, and more control over rebounding, steals, and blocks. New game broadcast presentation hosted by Ernie Johnson and Shaq O'Neal delivers commentary and analysis to the action.

NBA 2K14 System Requirements

PC Requirements

Minimum:

  • OS: Windows 7 64-bit, Windows 8.1 64-bit, Windows 10 64-bit
  • Processor: Intel Core 2 Duo or AMD Athlon 64 X2
  • Memory: 2 GB RAM
  • Graphics: DirectX 10.1 compatible (512 MB)
  • Storage: 50 GB available space

Recommended:

  • OS: Windows 7 64-bit, Windows 8.1 64-bit, Windows 10 64-bit
  • Processor: Intel Core i5 or AMD FX series
  • Memory: 4 GB RAM
  • Graphics: DirectX 11 compatible (2 GB)
  • Storage: 9 GB available space

Console Requirements

PlayStation 3 & Xbox 360:

  • Requires respective console
  • Ensure sufficient storage space for installation and updates

PlayStation 4 & Xbox One:

  • Requires respective console
  • Minimum 60 GB of hard drive space
