About the Game

Geometry Dash is a rhythm-based platformer game developed by Swedish game developer Robert Topala, also known as RobTop. It was first released on August 13, 2013, for iOS and Android, and later for Windows and macOS on December 22, 2014. In Geometry Dash, players control a customizable icon that navigates through music-based levels, avoiding obstacles like spikes and walls. The game features 26 main levels, each with its own unique soundtrack and challenges.

Geometry Dash System Requirements

PC Requirements

Minimum:

  • OS:Windows XP or later
  • Processor: 2.0+ GHz
  • Memory: 512 MB RAM
  • Graphics: OpenGL 2.0 support
  • Storage: OpenGL 2.0 support

Recommended:

  • OS:Windows 7 or later
  • Processor: 2.8 GHz Intel Core i3 or higher
