About the Game

Valorant is a free-to-play tactical first-person shooter developed and published by Riot Games. Combining precise gunplay with unique agent abilities, the game offers a competitive 5v5 experience where strategy, skill, and teamwork are key to victory. Featuring multiple game modes, frequent updates, and an active eSports scene, Valorant delivers an intense and rewarding shooter experience for players of all skill levels.

Valorant System Requirements

PC Requirements

Minimum (30 FPS):

  • OS: Windows 10/11 (64-bit)
  • Processor: Intel Core 2 Duo E8400 / AMD Athlon 200GE
  • Memory: 4 GB RAM
  • Graphics: Intel HD 3000 / AMD Radeon R5 200
  • DirectX: Version 11
  • Storage: 30 GB available space

Recommended (60+ FPS):

  • OS: Windows 10/11 (64-bit)
  • Processor: Intel Core i3-4150 / AMD Ryzen 3 1200
  • Memory: 4 GB RAM
  • Graphics: NVIDIA GeForce GT 730 / AMD Radeon R7 240
  • DirectX: Version 11
  • Storage: 30 GB available space

High-End (144+ FPS):

  • OS: Windows 10/11 (64-bit)
  • Processor: Intel Core i5-9400F / AMD Ryzen 5 2600X
  • Memory: 8 GB RAM
  • Graphics: NVIDIA GeForce GTX 1050 Ti / AMD Radeon R7 370
  • DirectX: Version 11
  • Storage: 30 GB available space

Availability

  • PC (Windows only)
